CommonHIT
====================================================================================
A set of files to reduce the boilerplate when designing HITs for the Amazon 
Mechanical Turk crowdsourcing platform.

 * commonhit.css
 	A default set of styles for HITs.
 * commonhit.js
 	Script for reducing boilerplate in HIT designs. 
 * lllight.js
 	A lightweight player for recording interactions with the YouTube player.


Status: Experimental
